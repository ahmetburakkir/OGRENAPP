# OGRENAPP

Geleceğini OGRENAPP ile şekillendir! Öğrenciler için özel olarak geliştirilmiş bu platform, oyunlaştırılmış özellikleri, gerçek dünya kariyer simülasyonları, karakter testleri ve hedef takibi ile geleceğin profesyonellerini bugünden hazırlıyor.

## Kurulum ve Çalıştırma

Projeyi lokalde çalıştırmak için:

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

## Teknolojiler
- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- React Router DOM
